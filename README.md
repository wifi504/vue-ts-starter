# Vue 3 + TypeScript + Vite 的初始化工程模板

每次开新项目都是从官方开始，但是发现有很多后续操作一模一样，干脆自己自定义一个好用的初始化模板

### 依赖概览

- 运行时依赖

| 名称 | 版本    | 描述            |
| ---- | ------- | --------------- |
| vue  | ^3.5.17 | Vue.js 框架环境 |

- 开发时依赖

| 名称               | 版本    | 描述 |
| ------------------ | ------- | ---- |
| @vitejs/plugin-vue | ^6.0.0  |      |
| @vue/tsconfig      | ^0.7.0  |      |
| typescript         | ~5.8.3  |      |
| vite               | ^7.0.4  |      |
| vue-tsc            | ^2.2.12 |      |
